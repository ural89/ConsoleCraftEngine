# Console Craft Engine
#### Written in C++
#### Will work in Linux and Windows
#### Box2D physics engine implemented
#### Now editor available: https://github.com/ural89/ConsoleCraftEditor
## Build:
#### cd ../../CraftShooterGame
#### mkdir build
#### cd build
#### cmake ..
#### make
